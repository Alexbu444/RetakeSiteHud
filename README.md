# RetakeSiteHud
Displays the current bombsite in a hud  
To be used with splewis version of the [retakes](https://github.com/splewis/csgo-retakes) plugin

# Installation
Add the `retakeshud.smx` into your `addons/sourcemod/plugins/` folder.
